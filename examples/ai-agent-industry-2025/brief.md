# 调研简报

## 主题
AI Agent 行业现状（2025）

## 调研范围
全面分析 AI Agent（智能体）行业的市场规模、技术演进、竞争格局和落地应用场景，重点关注 2024-2025 年的最新发展动态。

## 角色分配

### 1. 市场规模分析师（market-analyst）
**调研范围**：AI Agent 全球及中国市场规模、增长率、投融资情况、产业链结构
**必答问题**：
1. 2024-2025 年全球和中国 AI Agent 市场规模分别是多少？预计增长率？
2. 主要的投融资事件有哪些？资本市场对 AI Agent 的态度如何变化？
3. AI Agent 产业链上下游分别有哪些环节？各环节的市场占比如何？
**推荐 CLI**：Codex（数据验证）

### 2. 技术趋势分析师（tech-trend）
**调研范围**：AI Agent 核心技术架构、框架工具、技术演进路线、前沿研究方向
**必答问题**：
1. 当前主流的 AI Agent 技术架构是什么？（如 ReAct、Plan-and-Execute、Multi-Agent 等）
2. 主要的 Agent 开发框架有哪些？各自优劣和采用率如何？（如 LangChain、AutoGen、CrewAI、Dify 等）
3. 2025 年 AI Agent 技术的主要突破点和瓶颈是什么？
**推荐 CLI**：Gemini（技术分析）

### 3. 竞品格局分析师（competitor-analyst）
**调研范围**：全球及中国主要 AI Agent 产品/平台的对比分析、市场份额、差异化策略
**必答问题**：
1. 全球和中国的头部 AI Agent 产品/平台有哪些？各自定位和核心能力？
2. 大厂（OpenAI、Google、Anthropic、字节、阿里、百度等）的 Agent 战略布局如何？
3. 创业公司在 AI Agent 赛道的切入角度和竞争优势是什么？
**推荐 CLI**：Codex（竞品信息核实）

### 4. 应用场景分析师（app-scenario）
**调研范围**：AI Agent 在各行业的落地情况、典型案例、用户反馈、商业化路径
**必答问题**：
1. AI Agent 在哪些行业/场景已经实现规模化落地？效果如何？
2. 企业级 AI Agent 的典型部署模式和 ROI 情况？
3. AI Agent 商业化的主要挑战是什么？用户最大的痛点是什么？
**推荐 CLI**：Gemini（案例搜索）

## 质量标准
- 总分 ≥ 35/50，单项 ≥ 6/10
- 5 个评分维度：事实准确性、信源覆盖度、分析深度、内部一致性、完整性
